# vue-mobile

#### 介绍
基于 vue3.x + CompositionAPI + javascript + vite + vant + vue-router-next + pinia 的开源免费移动端模板，希望减少工作量，帮助大家实现快速开发移动端。
vue-mobile默认已加入 Rem布局适配，可在根目录 postcss.config.js 进行修改。

#### 使用说明

# 克隆项目
git clone https://gitee.com/swwzfy/vue-mobile.git

# 进入项目
cd vue-mobile

# 安装依赖
yarn

# 运行项目
yarn run dev

# 打包发布
yarn run build
